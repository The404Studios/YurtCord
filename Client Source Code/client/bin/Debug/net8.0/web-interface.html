Available Commands

/help - Display this help message

/users - List all connected users

/balance - Check your credit balance

/rooms - List all available chat rooms

/createroom name [description] - Create a new chat room

/join roomName - Join a chat room

/leave - Leave current room and return to lobby

/whisper username message - Send a private message

/transfer username amount [description] - Send credits to another user

/shout message - Post a message to the global shoutbox

/shoutbox - View recent shoutbox messages

/transactions - View your recent transactions

Gambling Commands

/gamble bet amount - Place a bet in the active pot

/pots - List all active gambling pots

/potinfo potId - Get detailed information about a specific pot

/gamblinghistory - View recent gambling winners

Casino Games

/dice amount [target] - Roll dice (target 2-12 is optional)

/flip amount choice - Flip a coin (choice: HEADS or TAILS)

/slots amount - Play the slot machine

/stats - See your gambling statistics

/leaderboard - View top gamblers
